Detailed Description

ROS2 node that subscribes to controller_wrench_topic and recieves F.F. instructions from S7Spaceship.

This class manages a subscriber for the S7Mothership, taking care of storing the current wrench msg.

Constructor for S7ForceSubscriber node.

Initializes the ROS2 node with the name "wrench_subscriber" and creates a subscriber on the topic "controller_wrench_stamped" containing F.F. instructions for the S7 arm, which is published to by a component of S7Spaceship.
